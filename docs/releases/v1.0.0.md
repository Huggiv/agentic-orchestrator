# Release v1.0.0

Release date: 2026-06-24

## Highlights

- Agent-aware orchestration is now first-class.
- Backend includes bundled Copilot agent definitions and agent discovery API.
- Frontend now supports explicit agent selection during run trigger.
- Running jobs and execution flow are more reliable and visible across reloads.
- History cards are redesigned for better readability and operational review.

## Included Changes

### Backend

- Added global Copilot agent copy step in backend Docker image build.
- Added selected_agent support to orchestration API payload and execution flow.
- Added GET /api/agents endpoint for dynamic agent listing.
- Improved commit step reporting so commit_changes is marked success when branch is already ahead.
- Expanded backend tests for agent endpoint, selected agent pass-through, and commit-ahead detection.

### Frontend

- Added Agent dropdown in Run tab, populated from backend agent list.
- Switched execution visibility to Executing tab after triggering a flow.
- Added shared flow-step component for consistent step display.
- Improved Executing tab polling and running job persistence.
- Refined History tab with:
  - View PR emphasis
  - Trigger timestamp badge
  - Duration badge near status
  - Changes and Usage expandable table
  - Artifacts expandable section

## API Updates

- New endpoint: GET /api/agents
- Updated orchestration payload field: selected_agent

## Notes

- This tag marks the first stable baseline for agentic orchestration workflows and dashboard UX.
